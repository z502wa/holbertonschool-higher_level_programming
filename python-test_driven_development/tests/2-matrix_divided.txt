Module: 2-matrix_divided
========================

Function: matrix_divided(matrix, div)
------------------------------------

This function divides all elements of a matrix by a number and returns a new matrix.

Examples:
---------
Normal cases:
    >>> matrix_divided([[1, 2, 3], [4, 5, 6]], 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
    
    >>> matrix_divided([[10, 20], [30, 40]], 2)
    [[5.0, 10.0], [15.0, 20.0]]
    
    >>> matrix_divided([[1.5, 2.5], [3.5, 4.5]], 0.5)
    [[3.0, 5.0], [7.0, 9.0]]

Edge cases:
    >>> matrix_divided([[1]], 1)
    [[1.0]]
    
    >>> matrix_divided([[-1, -2], [-3, -4]], -1)
    [[1.0, 2.0], [3.0, 4.0]]
